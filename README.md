# 🎱 LOTTO ORACLE v2.0

**19가지 방법론 앙상블 로또 예상번호 추출 시스템**

> 📱 PWA 지원 — 홈 화면에 설치하여 앱처럼 사용 가능

## ✨ 주요 기능

### 3계층 19가지 방법론 엔진
- **계층1 — 점수 엔진 (7가지)**: 빈도분석, 평균회귀, 최근성가중, 간격분석, 이월수, 멀티트렌드, 연관규칙
- **계층2 — 필터 파이프라인 (7가지)**: AC값, 합계범위, 홀짝밸런스, 대역분포, 스프레드, 끝수제한, 소수검증
- **계층3 — 5세트 다양화**: Hot Dominant, Cold Rebound, Consecutive+, Monte Carlo, Ensemble Vote

### 사용자 기능
- 📥 최신 당첨번호 직접 입력 (모바일 최적화)
- 🎯 5개 예상번호 세트 추출
- 📊 점수 분석 대시보드
- 🔬 상세 통계 분석
- ✅ 알고리즘 검증 로그
- 💾 데이터 자동 저장 (입력한 번호 유지)
- 📲 PWA 설치 (오프라인 지원)

## 🚀 사용 방법

1. **[번호입력]** 탭에서 최신 회차 당첨번호 입력
2. **[🚀 예상번호 추출]** 버튼 클릭
3. **[예상번호]** 탭에서 5개 세트 확인
4. **[검증로그]** 탭에서 알고리즘 작동 확인
